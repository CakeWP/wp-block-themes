FAQ

Frequently Asked Questions

01.

Is it a Classic or FSE theme?

GreenShift is a WordPress FSE theme and is based on blocks. You can use Site Editor to make custom archives, homepages, single pages, and others.

02.

Is it a free or premium theme?

GreenShift is WordPress free theme and will always be free. We plan to make premium child themes and design packs in the future, but core and all functionality will always be free for everyone.

03.

Can I get 100 web vitals without cache plugins?

GreenShift has conditional loading of all assets so there is a big chance that you will get a 100 score without any caching. But it would be best if you had good hosting with a small TTFB and pay attention to your content. We don’t recommend using heavy media on the first screen of the site. Also, replace all image files to webp or SVG format and avoid using external scripts and heavy plugins

04.

Do I need to use page builder plugins?

It’s not required. This page is built entirely on core blocks, however, to get maximum flexibility and functionality, we recommend using the free companion Greenshift plugin
