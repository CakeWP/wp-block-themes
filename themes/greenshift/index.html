FAQ

Frequently Asked Questions

01.

Is it Classic or FSE theme?

GreenShift is WordPress FSE theme and based on blocks. You can use Site editor to make custom archives, homepage, single pages and other.

02.

Is it free or premium theme?

GreenShift is WordPress free theme and will always be free. In future we plan to make premium child themes and design packs, but core and all functionality will be always free for everyone.

03.

Can I get 100 web vitals without cache plugins?

GreenShift has conditional loading of all assets so there is big chance that you will get 100 score without any caching. But you need to have good hosting with small TTFB and pay attention to your content. We don’t recommend to use heavy media in first screen of the site. Also, replace all image files to webp or svg format and avoid using external scripts and heavy plugins

04.

Do I need to use page builder plugins?

It’s not required. This page is built completely on core blocks, however to get maximum flexibility and functionality, we recommend to use free companion Greenshift plugin
