FullSite-Patterner

WordPress block theme for full site editing that can replace over 100 site patterns

CONCEPT

"FullSite-Patterner" is a WordPress block theme that you can easily install and replace the entire page with patterns, and freely replace headers, footers, and parts.

DOWNLOAD

A free WordPress block theme with ready-to-use sample sites and full site editing. Upgradable to the paid version with many functions and abundant block patterns.

INSTALL

Explains how to install and set up "FullSite-Petterner". All you have to do is set the category, URL, permalink, logo image, etc.

HOW TO USE

We will post how to install, change settings, and customize web site with the block theme "FullSite-Petterner".

PATTERN

1-column, 2-column, full-width, with or without background image, block that can replace the entire page, header, footer, content, etc. for each part.

PLUGIN

You can change the whole page, header, footer, etc. by replacing block patterns, and use the function that allows you to register patterns with other themes.

SEO PLUGIN

Plugin that can set meta tags, canonical, tags in head, breadcrumbs, structured data automatically or by input.

Q&A about "FullSite-patterner" installation, usage, customization, plug-in usage, future development schedule, etc.
